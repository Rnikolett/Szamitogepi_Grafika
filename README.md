# Szamitogepi_Grafika


A projekt egy játék, amelyben a focilabda gurul a padlón, és a játékosnak kerülnie kell a háromszög alakú piramist. A játékos vezérlése megváltoztatja a forgásirányt.

Segítség: H
Mozgás:
      gyorsítás: W
      Lassítás/hátra: S
      Forduljon balra: A
      Fordulj jobbra: D
Kamera:
      Fel: E
      Le: Q
      Nagyítás: +
      Kicsinyítés: -
Fény:
      Fényerö növelése: .
      Fényerö növelése: ,
      FX: L
Az akadály típusa:
      Piramis: 0
      Tüskék: 1
Kilépés: ESC
